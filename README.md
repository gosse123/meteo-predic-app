# 🌦️ Météo Ouaga Analytics

An educational web application demonstrating the application of mathematics to weather forecasting through statistical analysis, regression models, and interactive data visualization.

## 📖 About

**Météo Ouaga Analytics** is a university project developed to illustrate how mathematical concepts are applied in meteorology.

The application allows users to:

* Analyze historical weather data.
* Explore time series.
* Study seasonal variations.
* Compute statistical indicators.
* Visualize correlations between climate variables.
* Understand linear regression models used for prediction.

The project combines mathematics, statistics, machine learning, and modern web technologies into a single interactive platform.

## 🚀 Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
